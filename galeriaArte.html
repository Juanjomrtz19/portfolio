Aplicación web para una galería de arte

Tecnologías usadas

Descripción del problema

El proyecto consistía en hacer una aplicación web para una galería de arte digital

Había dos tipos de usuarios: comisarios y usuarios normales, los usuarios normales solo podían ver y poner comentarios; los comisarios debían poder crear, modificar y subir obras de arte (básicamente administraban el sitio).

Este fue mi primer proyecto web completo full stack y en el que aprendí una gran cantidad de conocimientos en las distintas tecnologías que usé.
